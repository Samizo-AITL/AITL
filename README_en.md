**AITL (All-in-Theory Logic)** is a theory-driven AI architecture that defines intelligent systems through a layered model:

- üß† **Logic Layer**: Semantic reasoning and intention understanding using LLMs  
- üîÅ **Control Layer**: Deterministic state transitions via FSM
