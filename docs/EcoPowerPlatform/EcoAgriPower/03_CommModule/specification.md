# 03_CommModule - Specification

## 概要

本モジュールは、Eco Agri Power Model の無線通信機能を担うLoRaベースの通信ユニットです。  
広域かつ低消費電力の通信を実現し、農地におけるセンシングデータの収集・送信を支援します。

---

## 主な仕様

| 項目             | 内容                                              |
|------------------|---------------------------------------------------|
| 通信方式         | LoRa（Sub-GHz帯域）                               |
| 周波数帯域       | 920 MHz帯（日本向け技適対応）                     |
| 通信距離         | 見通し最大 約5 km（実環境で 1〜2km 見込み）       |
| モード           | LoRaWAN Class A（将来対応）／独自プロトコル       |
| 通信制御         | ATコマンドまたはUARTベースAPI                     |
| 送信出力         | +14 dBm（日本国内出力制限準拠）                   |
| 消費電力         | 送信時 最大50mA、待機時 < 1µA（Deep Sleep）       |

---

## 通信プロトコル

- センサデータ形式：JSON風簡易構造  
- 送信周期：30分〜6時間（バッテリ状態と連動）  
- 複数センサ統合送信／分割送信対応

---

## システム連携

- AI SoCからUART制御で送信  
- LoRaモジュールの起動／スリープ管理はSoCが制御  
- GUIツールから通信ログ取得可能（PoC用途）

---

## 教育・PoC利用

- 無償提供予定のLoRa受信機と組み合わせた受信確認ツールあり  
- 学校・自治体用簡易ゲートウェイ構成も評価中

---

## 今後の拡張

- GNSS連携（時刻同期）対応のファームウェア更新予定（v1.1）  
- AES暗号通信対応（LoRaWAN向け）を将来提供予定
