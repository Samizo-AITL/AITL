# 07_EvalKit - Specification

## 概要

本モジュールは、Eco Agri Power Model の評価および教育用キットです。  
評価ボード、ソーラーパネル、バッテリ、GUI、教材一式をセットにし、教育機関やPoCでの利用を想定しています。

---

## 主な仕様

| 項目               | 内容                                               |
|--------------------|----------------------------------------------------|
| 評価ボード         | Eco Agri Power各モジュール搭載評価用PCB           |
| 電源               | 2Wソーラーパネル + 3.2V/1500mAh LFPバッテリ         |
| GUIツール          | PythonベースのGUIソフトウェア                       |
| 教材               | センサ・AI・通信の基礎教材・演習問題付き             |
| 付属品             | USBケーブル、センサ拡張モジュール、取扱説明書        |

---

## 教育・PoC利用

- 学校・自治体向けに無償提供予定  
- センサデータ取得とAI推論の実演が可能  
- 電源管理、通信状態の実践的理解支援  

---

## 今後の拡張

- 拡張センサキット追加予定（v1.2以降）  
- オンライン教材・動画チュートリアル連携
