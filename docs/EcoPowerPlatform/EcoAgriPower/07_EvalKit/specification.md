# 06_Firmware - Specification

## 概要

本モジュールは、Eco Agri Power Model のマイコン上で動作するファームウェアです。  
センサデータの取得、AI推論の実行、LoRa通信制御、低電力管理など全体制御を担います。

---

## 主な仕様

| 項目               | 内容                                           |
|--------------------|------------------------------------------------|
| 対応MCU            | 65nm CMOS RISC Core搭載AI SoC                   |
| 開発言語           | C言語（組み込み）、Pythonスクリプト（教育用）  |
| 主な機能           | センサデータ取得（I²C/SPI/ADC）                 |
|                    | AI推論処理                                     |
|                    | LoRa通信制御                                   |
|                    | 低電力スリープ管理                             |
|                    | GUIとの同期通信（UART）                         |

---

## ソフトウェア構成

- センサ管理モジュール  
- AI推論制御モジュール  
- 通信制御モジュール  
- 電源管理モジュール  
- GUI連携通信モジュール  

---

## 教育・PoC利用

- 学習用AIモデルのブラックボックス化  
- GUI操作と連動した動作確認サンプルコード付き  
- 電力消費・性能の教育用計測ツール対応

---

## 今後の拡張

- OTA（Over-The-Air）アップデート対応（v1.3以降予定）  
- 複数センサ動的切替対応  
- ファームウェア自動生成サポート（SystemDK連携）
