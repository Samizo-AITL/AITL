# 02_PowerModule - Specification

## 概要

本モジュールは、Eco Agri Power Modelの自律動作を支える電源ユニットです。  
低照度環境下でも安定稼働できるよう、2Wソーラーパネル、MPPT制御回路、LFP電池を組み合わせた構成となっています。

---

## 主な仕様

| 項目               | 内容                                           |
|--------------------|------------------------------------------------|
| 電源入力           | 単結晶ソーラーパネル（最大出力 2W）           |
| MPPT制御回路       | 効率最大化制御、起動電圧追従                   |
| バッテリ種類       | LiFePO₄（LFP）3.2V / 1500mAh                    |
| 出力電圧           | 3.3V DC 安定化（オンボードレギュレータ）       |
| バッテリ充電方式   | ソーラーパネルによる直接充電（定電圧制御）     |
| 保護機能           | 過電圧・過放電・逆流防止                        |
| 推定稼働時間       | 曇天・無日照状態で連続約48時間稼働可能         |

---

## 構成ブロック

- 高効率DC-DC昇圧回路  
- MPPTアルゴリズム搭載制御IC  
- 保護付きLFP充電回路  
- 電圧・電流監視モジュール（センサ側提供予定）

---

## システム連携

- 電圧モニタ値をAI SoCにI²C経由で提供  
- 電力残量に応じた制御動作（AI推論間隔調整など）

---

## 教育・PoC利用

- ソーラー/バッテリ状態のGUI可視化対応  
- モジュール単体評価基板あり（評価キット同梱）

---

## 今後の拡張

- 日射量・温度を用いた発電予測連動制御（v1.2対応検討）  
- 複数電源入力対応（外部DC/AC変換器接続）オプション予定
