# 05_GUI - Specification

## 概要

本モジュールは、Eco Agri Power Model の教育・評価用GUIツールです。  
Pythonベースで開発され、センサデータの可視化やAI推論結果の表示、通信状態のモニタリングを行います。

---

## 主な仕様

| 項目              | 内容                                           |
|-------------------|------------------------------------------------|
| プラットフォーム  | Windows, macOS, Linux対応                      |
| 開発言語          | Python 3.x                                     |
| GUIフレームワーク | PyQt5 / Tkinter（将来的にWebベース対応予定）  |
| 通信方式          | USBシリアル（UART）、UART-to-USBアダプタ対応  |
| 機能              | センサ値のリアルタイム表示                      |
|                   | AI推論結果グラフ表示                            |
|                   | LoRa通信状態表示                                |
|                   | 設定値の読み書き                                |
|                   | データログ保存（CSV形式）                        |

---

## システム連携

- ファームウェアと双方向通信可能（UART経由）  
- PoCDKおよびSystemDKのGUI設計支援ツールと連携予定  
- 教育用途での操作マニュアル・チュートリアル付き

---

## 教育・PoC利用

- センサデータをリアルタイムで確認可能  
- AIモデルの動作を視覚的に理解しやすい設計  
- データのエクスポート・解析用CSV生成機能あり

---

## 今後の拡張

- WebベースGUIへの移行検討（v2.0以降）  
- スマホアプリ連携対応（Bluetooth LE 経由）  
- 多言語対応（日本語・英語・中国語）
