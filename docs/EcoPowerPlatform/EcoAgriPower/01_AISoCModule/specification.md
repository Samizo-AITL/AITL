# 01_AISoCModule - Specification

## 概要

本モジュールは、Eco Agri Power Modelにおける中核的な制御・処理ユニットです。  
RISCベースのプロセッサと軽量AI推論エンジンを組み合わせ、センサデータの取得・処理・予測を担います。

---

## 主な仕様

| 項目             | 内容                                               |
|------------------|----------------------------------------------------|
| プロセス技術     | 65nm CMOS                                           |
| プロセッサ構成   | RISC Core（32bit） + 軽量AIエンジン                 |
| AI推論機能       | 温湿度の時系列予測、しきい値異常検知                |
| 内蔵IF           | I²C, SPI, ADC（8bit/10bit）、PWM                    |
| メモリ構成       | SRAM 32KB、Flash 128KB                              |
| 動作周波数       | 16〜48 MHz（低電力モード対応）                      |
| 消費電力         | 動作時 5〜15mW、スリープ時 < 1mW                    |

---

## 機能ブロック構成

- AI推論エンジン（温湿度データ予測、異常検知）
- センサI/F（I²C/SPI/ADC経由）
- クロック・電源制御機能（低電力制御）
- LoRa通信制御部との連携IF

---

## ソフトウェア連携

- ファームウェアでAI推論・センサ処理を統合制御  
- GUIモジュール経由で推論結果の可視化対応  
- SystemDKによる自動検証スクリプト対応予定

---

## 教育・PoC利用

- AI推論ブロックのブラックボックス提供（教育用）  
- Python GUIから動作確認可能（推論入力／出力確認）

---

## 今後の拡張

- モデル再学習対応の簡易オンライン更新機構（v1.2以降予定）  
- データログ保存機能（センサ・推論出力のFlash記録）
