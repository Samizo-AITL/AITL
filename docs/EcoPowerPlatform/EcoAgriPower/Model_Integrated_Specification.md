# Eco Agri Power Model - 統合仕様書 v0.1

## 1. 概要

Eco Agri Power Modelは、農業向けに設計された超低消費電力の環境センシングIoTプラットフォームです。  
AI推論搭載のSoCを核に、ソーラー電源、LoRa通信、各種環境センサ、教育用GUIなどを統合し、地域PoCに向けて開発されています。

---

## 2. モジュール構成

| No. | モジュール名         | 主な仕様概要                                               |
|-----|----------------------|------------------------------------------------------------|
| 01  | AI SoCモジュール      | 65nm CMOS RISC Core + 軽量AI推論、I²C/SPI/ADC内蔵           |
| 02  | 電源モジュール        | 2Wソーラー + MPPT制御 + LFP電池（3.2V/1500mAh）              |
| 03  | 通信モジュール        | LoRa 920MHz帯、最大5km通信、LoRaWAN Class A対応予定         |
| 04  | センサI/F            | 温湿度・日射・CO₂など多様なセンサ接続、I²C/SPI/ADC対応      |
| 05  | GUIモジュール         | Python GUI、リアルタイムデータ可視化、教育用途対応          |
| 06  | ファームウェア        | センサ制御・AI推論・LoRa通信制御・低電力管理                |
| 07  | 評価・教育キット      | 評価ボード一式＋教材＋GUI、教育機関向け無償提供予定          |
| 08  | 拡張インターフェース  | GNSS・土壌圧力センサ等拡張I/F、将来の追加センサ対応         |
| 09  | PoCデータ連携        | 通信ログ・AI判定記録・学習データ収集、地域PoCデータ管理基盤 |

---

## 3. 各モジュール詳細

### 01_AISoCModule  
- RISCベースAI SoC。温湿度予測と異常検知対応。  
- I²C/SPI/ADCのセンサインタフェース内蔵。

### 02_PowerModule  
- 2Wソーラーパネル＋MPPT制御＋LFPバッテリ。  
- 低照度環境でも約48時間連続稼働可能。

### 03_CommModule  
- LoRa通信920MHz帯。最大5km見通し距離。  
- LoRaWAN Class A対応予定。

### 04_SensorInterface  
- 多種センサ対応のI²C/SPI/ADCインタフェース。  
- プラグアンドプレイ検出機能搭載。

### 05_GUI  
- PythonベースGUI。リアルタイム可視化と操作性重視。  
- 教育用チュートリアル付き。

### 06_Firmware  
- MCU上でセンサ制御、AI推論、通信、低電力管理を統合。  
- GUI連携・教育用サンプルコード付き。

### 07_EvalKit  
- 評価用ボード・ソーラー・バッテリ・GUI・教材一式。  
- 教育機関・PoC向けに無償提供予定。

### 08_ExtInterface  
- GNSS、土壌圧力センサなど拡張用I/F。  
- 将来的にホットプラグ・産業用バス対応検討。

### 09_PoCDataLink  
- 地域PoCの通信ログ・AI判定データ収集管理基盤。  
- クラウド連携・データ解析支援予定。
